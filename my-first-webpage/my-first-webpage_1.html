    HTML '일반태그' 모음

    일반 HTML 태그는 크게 여러 그룹으로 나눌 수 있다.
일반 태그는 시작 태그와 닫는 태그로 구성되며, 그 사이에 콘텐츠를 담는다.
아래는 HTML5를 기준으로 한 주요 일반 태그들이다

      문서 구조 및 기본

    • <!DOCTYPE html> : "이 문서는 HTML5로 작성되었어요!"라고 브라우저에게 알려주는 선언문
    • <html> : HTML 문서의 모든 내용을 감싸는 최상위 요소
    • <head> : 사용자에게는 보이지 않지만 중요한 정보들이 들어가는 곳, 문서의 메타데이터를 포함하며 브라우저에 정보를 전달
    • <title> : 웹 페이지의 제목을 정의합니다. 브라우저 탭에 표시될 제목
    • <body> : 웹 페이지에 표시되는 모든 콘텐츠(본문)를 담는 태그

      • 텍스트 및 콘텐츠

    • <h1> ~ <h6> : 제목을 정의하는 태그 <h1>이 가장 중요한 제목이며, <h6>가 가장 덜 중요한 제목이다. (그냥 크기차이)
    • <p> : 문단을 정의(문단 영역을 구분해줌)
    • <q> : 짧은 인용문을 문장 안에 포함 (일반적인 따옴표 역할)
    • <a> : 하이퍼링크(클릭시 지정된 위치이동)를 만들어줌
    • <span> : 인라인 요소로, 텍스트의 일부분만 감쌀 때 사용되며 특별한 의미는 없음(영역 구분용도)
    • <div> : 블록 레벨 컨테이너, 큰 영역을 나누는 용도. 웹페이지의 섹션을 구분할 때 사용되며 특별한 의미는 없음(영역 구분용도)
    • <strong> : 중요한 텍스트를 강조(굵게 표시)
    • <em> : 텍스트의 중요성을 강조(기울게 표시)
    • <code> : 짧은 코드나 명령어를 인라인으로 표시
    • <blockquote> : 긴 인용문을 독립된 블록으로 표시(들여쓰기 + 문단영역 구분)
    • <pre> : 긴 코드 블록을 원본 형태 그대로 표시 (공백과 줄바꿈 유지)
    • <cite> : 인용 출처 표시

      • 시맨틱 구조

      HTML5에서 도입된 시맨틱 태그는 웹 문서의 구조와 의미를 명확하게 한다.
    • <header> : 페이지나 섹션의 머리글(소개 콘텐츠)을 정의
    • <nav> : 내비게이션 링크를 포함하는 영역을 정의
    • <main> : 문서의 주요 콘텐츠를 정의
    • <article> : 독립적인 콘텐츠를 정의합니다 (예: 블로그 게시물, 뉴스 기사)
    • <section> : 문서의 독립적인 섹션을 정의
    • <aside> : 페이지의 주요 콘텐츠와 간접적으로 관련된 콘텐츠(사이드바 등)를 정의
    • <footer> : 페이지나 섹션의 바닥글을 정의
    • <figure>, <figcaption> : 이미지, 다이어그램, 코드 조각 등과 그에 대한 캡션을 정의

      • 멀티미디어 관련

    • <audio> : 오디오 콘텐츠를 삽입
    • <video> : 비디오 콘텐츠를 삽입
    • <iframe> : 다른 HTML 문서를 현재 문서에 삽입

      • 폼 관련

    • <form> : 사용자의 입력을 받는 양식 정의
    • <label>> : 폼 컨트롤의 설명을 정의
    • <select>, <option> : 드롭다운 목록을 만든다.
    • <textarea> : 여러 줄의 텍스트 입력 필드를 만든다.
    • <button> : 클릭 가능한 버튼을 만든다.
    • <fieldset>, <legend> : 폼에서 관련 컨트롤들을 그룹화한다.

      • 표(테이블) 관련

    • <table> : 전체 테이블을 감싸는 컨테이너
    • <thead> : 테이블 머리글 영역 (제목 행들을 묶어줌)
    • <tbody> : 테이블 본문 영역 (실제 데이터들이 들어가는 곳)
    • <tfoot> : 테이블의 바닥글 부분 정의
    • <tr> : 테이블의 한 행
    • <th> : 제목 셀 (굵게 표시되고 가운데 정렬)
    • <td> : 일반 데이터 셀
    • <caption> : 테이블의 캡션을 정의

      • 목록 관련

    • <ul> : 순서 없는 목록 (순서가 중요하지 않은 목록을 만들 때 사용)
    • <ol> : 순서 있는 목록 (순서가 중요한 목록을 만들 때 사용)
    • <li> : 목록의 각 항목 (<ul>이나 <ol> 안에서만 사용할 수 있음, 단일사용 절대불가)*점 목록*

    HTML "단일태그" 모음

    HTML에서 단일 태그는 콘텐츠를 가질 수 없는 '빈 요소(void element)'를 의미
이러한 단일태그는 여는 태그만 있고 닫는 태그가 필요 없음
HTML에서는 편의상 끝에 슬래시(/)를 붙이지 않아도 되지만,
XHTML과의 호환성을 위해 붙이기도 함

      시각 및 구조

    • <br> : 줄바꿈 (엔터의 역할)
    • <hr> : 주제 전환이나 내용 구분을 위한 의미론적 수평 구분선
    • <img> : 이미지를 삽입

      • 멀티미디어 관련

    • <embed> : 외부 콘텐츠(플러그인, 비디오 등)를 삽입
    • <source> : <audio>, <video>, <picture> 요소의 미디어 소스를 지정
    • <track> : <video> 또는 <audio> 태그 내부에 사용하여, 자막, 캡션, 챕터 제목 등과 같은 미디어의 텍스트 트랙을 지정

      • 문서 메타 정보(데이터) 관련

      <head> 태그 안에 위치하여 문서의 메타데이터를 정의
    • <link> : 외부 리소스(주로 CSS 파일)를 연결
    • <meta> : 문서에 대한 메타데이터(예: 문자 집합, 뷰포트 설정, 검색 엔진 키워드 등)를 정의
    • <base> : 문서 내의 모든 상대 URL에 대한 기본 URL을 지정
    • <area> : <map> 태그 내부에 사용하여, 이미지 맵에서 클릭 가능한 영역을 정의

      • 폼 관련

    • <input> : 사용자가 데이터를 입력할 수 있는 필드를 정의
    • <param> : <object> 요소에 매개변수를 전달

      • 표 관련

    • <col> : <colgroup> 요소 내의 열에 대한 속성을 지정

      • 기타

    • <wbr> : "단어 분리(Word Break)"를 의미하며, 텍스트가 길어질 경우 줄 바꿈이 가능한 위치를 브라우저에 제안

    웹에 내가 적은 태그 그대로 보이게 하기

    & lt;내용& gt; // & lt;/내용 & gt;"
    (단일태그 아님 즉, 열고 닫아야함, 두번째 &앞에 내용넣기)

    HTML "속성" 모음

    HTML 속성은 HTML 요소의 시작 태그에 추가 정보를 제공하는 데 사용됩니다. 속성은 속성명="값"의 형태로 작성되며, 요소의 동작 방식이나 표시되는 모습을 제어합니다. 
HTML 속성은 크게 **전역 속성(Global Attributes)**과 **특정 태그 전용 속성(Element-specific Attributes)**으로 나눌 수 있습니다.

      전역 속성 (Global Attributes)

      대부분의 HTML 요소에 공통으로 사용할 수 있는 속성
    • id : 문서 내에서 요소를 고유하게 식별하는 데 사용하며 주로 CSS로 특정 요소를 스타일링하거나 자바스크립트로 조작할 때 사용한다.
    • class : 요소에 하나 이상의 클래스 이름을 지정하며 여러 요소에 동일한 클래스 이름을 부여하여 CSS로 한 번에 스타일을 적용하거나 자바스크립트로 제어할 때 사용한다.
    • style : 요소에 인라인(inline) CSS 스타일을 직접 적용한다. style="color: red; font-size: 16px;"와 같이 사용한다.
    • title : 요소에 대한 추가 정보를 제공한다. 마우스를 요소 위에 올리면 도구 설명(tooltip) 형태로 나타난다.
    • lang : 요소의 언어를 지정합니다. lang="ko"는 한국어, lang="en"은 영어를 의미. 주로 <html> 태그에 사용한다.
    • dir : 텍스트의 방향을 지정한다. ltr(왼쪽에서 오른쪽) 또는 rtl(오른쪽에서 왼쪽) 값을 가진다.
    • data-* : 사용자 지정 데이터를 저장하기 위해 사용된다. 자바스크립트에서 접근할 수 있는 사용자 정의 속성이다.
    • tabindex : 요소의 탭 순서를 지정한다. 키보드 탭 키로 요소에 접근하는 순서를 제어한다.
    • hidden : 요소가 렌더링되지 않도록 숨긴다.
    • contenteditable : 사용자가 요소의 콘텐츠를 편집할 수 있는지 지정한다.

      • + 추가 전역속성(조금 더 세부적인 전역속성)
    • accesskey : 요소에 포커스를 맞추는 단축키를 지정한다.
    • draggable : 요소를 드래그할 수 있는지 지정하며 주로 자바스크립트 드래그 앤 드롭 API와 함께 사용된다.
    • dropzone : 드래그된 요소가 드롭될 때의 동작을 지정한다.
    • spellcheck : 요소의 맞춤법 검사 여부를 지정한다.
    • translate : 요소의 내용이 페이지가 번역될 때 번역될지 여부를 지정한다.
    • autofocus : 페이지 로드 시 해당 요소에 자동으로 포커스를 맞춘다.
    • inert : 요소와 그 하위 요소에 대한 사용자 상호작용(클릭, 포커스 등)을 비활성화한다.

      • 특정 태그 전용 속성

      특정 태그에서만 사용되거나, 특정 태그에 따라 의미가 달라지는 속성
      하이퍼링크 <a>, <area>
    • href : 링크의 목적지 URL을 지정한다.

    • target : 링크된 문서를 여는 방법을 지정한다.
      1. _self(현재 창에 연다.(기본값))
      2. _blank(새 창이나 탭에 연다.)
      3. _parent(부모 프레임에 연다.)
      4. _top(최상위 프레임에 연다.)
      등의 값을 가진다.

    • rel : 현재 문서와 링크된 문서 사이의 관계를 지정합니다. nofollow, noopener 등의 값을 가진다.
    • download: 링크를 클릭했을 때 연결된 리소스를 다운로드하도록 지정한다.
    • hreflang: 링크된 문서의 언어를 지정한다.
    • media: 링크된 문서가 최적화된 미디어/장치 유형을 지정한다.

      • 이미지 <img>
    • src: 표시할 이미지의 경로를 지정한다
    • alt : 이미지를 볼 수 없을 때 대신 보여줄 텍스트이다. 시각장애인을 위한 스크린 리더나 이미지 로딩 실패 시에 사용된다.
    • width, height : 이미지의 너비와 높이를 픽셀 단위로 지정한다.
    • loading : 이미지를 로드하는 방식을 지정한다. (lazy 또는 eager)

      • 폼 요소 <input>, <form> 등
    • type : <input> 요소의 종류(텍스트, 비밀번호, 라디오 버튼, 체크박스 등)를 지정한다.
    • name : 폼 데이터를 전송할 때, 데이터의 이름을 지정한다.
    • value : 폼 요소의 초기값을 지정한다.
    • placeholder : 입력 필드에 입력할 값에 대한 힌트를 제공한다.
    • required : 해당 필드를 필수 항목으로 지정한다.
    • action : 폼이 제출될 때 데이터를 보낼 URL을 지정한다.
    • method : 폼 데이터를 전송할 HTTP 메서드(GET 또는 POST)를 지정한다

      • + 추가 폼 요소 속성(조금 더 세부적인 폼 속성)
    • autocomplete : 폼 필드의 자동완성 기능 여부를 지정한다.
    • form : <input>, <buttone> 등 폼 요소가 속한 <form>의 id를 지정하여, <form> 태그 외부에 있어도 폼에 속하게 만든다.
    • list : <datalist> 요소의 id를 참조하여 입력 필드에 사전 정의된 옵션을 제공한다.
    • pattern : 정규 표현식을 사용하여 입력 필드의 유효성을 검사한다.
    • maxlength, minlength : 입력할 수 있는 문자의 최대/최소 길이를 지정한다.
    • disabled : 폼 요소(버튼, 입력 필드 등)를 비활성화한다.
    • multiple : 여러 파일이나 항목을 선택할 수 있게 한다.

      • 멀티미디어 <video>, <audio>, <track> 등
    • autoplay : 페이지 로드 시 자동으로 재생되도록 한다.
    • controls : 재생, 일시정지, 볼륨 조절 등 기본 제어판을 표시한다.
    • loop : 반복 재생을 설정한다.
    • muted : 음소거 상태로 시작한다.
    • poster : 동영상 재생 전 표시될 이미지를 지정한다.
    • preload : 미디어 파일의 사전 로딩 방식을 지정한다.
    • kind : <track> 요소의 종류(자막, 캡션, 설명 등)를 지정한다.

      • 테이블 <table>
    • colspan : 셀이 차지할 열의 수를 지정한다.
    • rowspan : 셀이 차지할 행의 수를 지정한다.
    • headers: <th> 요소의 id를 참조하여 데이터 셀(<td>)과 헤더 셀 간의 관계를 정의한다.
