HTML "일반태그" 모음

일반 HTML 태그는 크게 여러 그룹으로 나눌 수 있다.
일반 태그는 시작 태그와 닫는 태그로 구성되며, 그 사이에 콘텐츠를 담는다.
아래는 HTML5를 기준으로 한 주요 일반 태그들이다

    문서 구조 및 기본

  • <html> : HTML 문서의 최상위(루트) 요소
  • <head> : 문서의 메타데이터를 포함하며, 브라우저에 정보를 전달
  • <title> : 웹 페이지의 제목을 정의합니다. 브라우저 탭에 표시
  • <body> : 웹 페이지에 표시되는 모든 콘텐츠(본문)를 담는 태그

    • 텍스트 및 콘텐츠

  • <h1> ~ <h6> : 제목을 정의하는 태그 <h1>이 가장 중요한 제목이며, <h6>가 가장 덜 중요한 제목이다. (그냥 크기차이)
  • <p> : 문단을 정의(문단 영역을 구분해줌)
  • <a> : 하이퍼링크(클릭시 지정된 위치이동)를 만들어줌
  • <span> : 인라인 요소를 묶는 데 사용되며 특별한 의미는 없음(영역 구분용도)
  • <div> : 블록 레벨 요소를 묶는 데 사용되며 특별한 의미는 없음(영역 구분용도)
  • <strong> : 중요한 텍스트를 강조(굵게 표시)
  • <em> : 텍스트의 중요성을 강조(기울게 표시)
  • <blockquote> : 인용 블록을 정의(들여쓰기 + 문단영역 구분)

    • 시맨틱 구조

    HTML5에서 도입된 시맨틱 태그는 웹 문서의 구조와 의미를 명확하게 한다.
  • <header> : 페이지나 섹션의 머리글(소개 콘텐츠)을 정의
  • <nav> : 내비게이션 링크를 포함하는 영역을 정의
  • <main> : 문서의 주요 콘텐츠를 정의
  • <article> : 독립적인 콘텐츠를 정의합니다 (예: 블로그 게시물, 뉴스 기사)
  • <section> : 문서의 독립적인 섹션을 정의
  • <aside> : 페이지의 주요 콘텐츠와 간접적으로 관련된 콘텐츠(사이드바 등)를 정의
  • <footer> : 페이지나 섹션의 바닥글을 정의
  • <figure>, <figcaption> : 이미지, 다이어그램, 코드 조각 등과 그에 대한 캡션을 정의

    • 멀티미디어 관련

  • <audio> : 오디오 콘텐츠를 삽입
  • <video> : 비디오 콘텐츠를 삽입
  • <iframe> : 다른 HTML 문서를 현재 문서에 삽입

    • 폼 관련

  • <form> : 사용자의 입력을 받는 양식 정의
  • <label>> : 폼 컨트롤의 설명을 정의
  • <select>, <option> : 드롭다운 목록을 만든다.
  • <textarea> : 여러 줄의 텍스트 입력 필드를 만든다.
  • <button> : 클릭 가능한 버튼을 만든다.
  • <fieldset>, <legend> : 폼에서 관련 컨트롤들을 그룹화한다.

    • 표(테이블) 관련

  • <table> : 표를 정의.
  • <thead> : 테이블의 헤더 부분 정의
  • <tbody> : 테이블의 본문 부분 정의
  • <tfoot> : 테이블의 바닥글 부분 정의
  • <tr> : 테이블의 한 행을 정의
  • <th> : 테이블의 헤더 셀을 정의
  • <td> : 테이블의 일반 데이터 셀을 정의
  • <caption> : 테이블의 캡션을 정의

    • 목록 관련

  • <ul> : 표를 정의.
  • <ol> : 테이블의 헤더 부분 정의
  • <li> : 테이블의 본문 부분 정의
  • <dl>, <dt>, <dd> : 정의 목록을 정의하며, <dt>는 용어, <dd>는 용어에 대한 설명을 정의한다.

HTML "단일태그" 모음

HTML에서 단일 태그는 콘텐츠를 가질 수 없는 '빈 요소(void element)'를 의미
이러한 단일태그는 여는 태그만 있고 닫는 태그가 필요 없음
HTML에서는 편의상 끝에 슬래시(/)를 붙이지 않아도 되지만,
XHTML과의 호환성을 위해 붙이기도 함

    시각 및 구조

  • <br> : 줄바꿈 (엔터의 역할)
  • <hr> : 수평선을 그어 내용을 구분
  • <img> : 이미지를 삽입

    • 멀티미디어 관련

  • <embed> : 외부 콘텐츠(플러그인, 비디오 등)를 삽입
  • <source> : <audio>, <video>, <picture> 요소의 미디어 소스를 지정
  • <track> : <video> 또는 <audio> 태그 내부에 사용하여, 자막, 캡션, 챕터 제목 등과 같은 미디어의 텍스트 트랙을 지정

    • 문서 메타 정보(데이터) 관련

    <head> 태그 안에 위치하여 문서의 메타데이터를 정의
  • <link> : 외부 리소스(주로 CSS 파일)를 연결
  • <meta> : 문서에 대한 메타데이터(예: 문자 집합, 뷰포트 설정, 검색 엔진 키워드 등)를 정의
  • <base> : 문서 내의 모든 상대 URL에 대한 기본 URL을 지정
  • <area> : <map> 태그 내부에 사용하여, 이미지 맵에서 클릭 가능한 영역을 정의

    • 폼 관련

  • <input> : 사용자가 데이터를 입력할 수 있는 필드를 정의
  • <param> : <object> 요소에 매개변수를 전달

    • 표 관련

  • <col> : <colgroup> 요소 내의 열에 대한 속성을 지정

    • 기타

  • <wbr> : "단어 분리(Word Break)"를 의미하며, 텍스트가 길어질 경우 줄 바꿈이 가능한 위치를 브라우저에 제안

웹에 내가 적은 태그 그대로 보이게 하기

& lt;내용& gt; // & lt;/내용 & gt;"
(단일태그 아님 즉, 열고 닫아야함, 두번째 &앞에 내용넣기)
